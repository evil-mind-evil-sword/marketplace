# emes Marketplace

**Plugin distribution for Claude Code.** Browse and install emes plugins.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add evil-mind-evil-sword/marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **alice** | Quality gate for Claude Code. Every exit requires alice review. Skills: reviewing, researching, issue-tracking, technical-writing, bib-managing. |

## Installing Plugins

```bash
/plugin install alice@emes
```

## Updating

```bash
/plugin marketplace update emes
```
