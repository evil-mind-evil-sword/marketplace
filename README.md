# evil-mind-evil-sword Marketplace

Claude Code plugin marketplace for evil-mind-evil-sword tools.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add evil-mind-evil-sword/marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **idle** | Multi-model development agents for exploration, research, reasoning, documentation, and review |

## Installing Plugins

```bash
/plugin install idle@evil-sword
```

## Updating

```bash
/plugin marketplace update evil-sword
```
