# emes Marketplace

**Plugin distribution for Claude Code.**

The official plugin marketplace for the evil-mind-evil-sword (emes) project.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add evil-mind-evil-sword/marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **idle** | Agentic harness for Claude Code. Long-running loops, multi-model consensus, message passing. |

## Installing Plugins

```bash
/plugin install idle@emes
```

## Updating

```bash
/plugin marketplace update emes
```
